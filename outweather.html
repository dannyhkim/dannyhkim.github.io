OutWeather

OutWeather is a web application built to help users determine what to wear, and what accessories to bring with them based on real-time weather conditions. By scraping weather data using an API and displaying the data using Flask App, the web app then utilizes the information to formulate an ideal outfit for the user to wear. Main factors affecting outfit suggestions include temperature, precipitation, and the overall season. OutWeather's user interface was designed to be very minimal, but seamless, aesthetically pleasing, and intuitive.
