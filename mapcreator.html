Map Creator

Map Creator was a game developed allowing users to save, load, and play their own custom maps. Users can select from a variety of block types, inventory, and develop many mazes and game solutions using the tools provided in the interface's side menu. The overall objective of the game is for the Heart Player to navigate through the maze, collect cash in order to go through the ATM's, and grab all the keys to be able to step onto the purple end block and win! This game was created using the MVC pattern in Java, alongside the AWT.
